# feature: 压缩包资源阅读

> 日期: 2026-07-17 | 类型: feature | 状态: ✅ 已完成

## 描述

将常用漫画压缩包作为独立资源和可预览文件处理，行为对齐 PDF：源文件浏览可直接预览、可多选添加到资源库、首页可展示缩略图并进入 Viewer。第一版支持 `zip/cbz/7z`，`rar/cbr` 受当前依赖限制仅识别为压缩包资源并显示明确不支持提示。

## 验收标准

- [x] `zip/cbz/7z` 可作为 `ResourceType.ARCHIVE` 添加到资源库
- [x] 首页 Archive 资源点击进入 Viewer
- [x] 源文件浏览中 `zip/cbz/7z` 可生成首图缩略图
- [x] 源文件浏览中 `zip/cbz/7z` 点击可进入 Viewer 阅读
- [x] Viewer 按压缩包内部图片自然顺序加载页面
- [x] `rar/cbr` 显示明确不支持提示，不静默失败
- [x] 不把压缩包混入文件夹内部 FlatGrid/Gallery 内容流
