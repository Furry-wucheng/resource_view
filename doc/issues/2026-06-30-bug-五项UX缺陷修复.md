# bug: 五项UX缺陷修复 — 批量标签/已入库复选框/跨章导航/详情弹窗/散落文件

> 日期: 2026-06-30 | 类型: bug | 状态: ✅ 已完成

## 现象

1. **首页资源库批量只有删除功能，不能批量添加标签**
2. **ResourcePicker 已入库节点同时显示绿色勾号 badge 和复选框**
3. **章节/章廊浏览无法从首/末页继续滑动跳转到上/下一章**
4. **首页长按详情弹窗缺少删除选项和快捷添加标签**
5. **章节散落文件点击后无法翻页浏览同目录其他文件**

## 复现步骤

1. 首页 → 多选模式 → 底部仅有"批量删除"
2. 文件浏览器 → 扫描入库弹窗 → 已入库文件夹同时显示勾号 badge + 复选框
3. 章节模式资源 → 翻到最后一页继续右滑 → 不切换到下一章
4. 首页 → 长按资源 → 详情弹窗无删除按钮、无快捷新建标签输入
5. 章节列表 → 点击散落图片文件 → 进入查看器但无法左右滑动翻看同级文件

## 期望效果

1. 多选底部栏同时有"批量添加标签"和"批量删除"两个按钮
2. 已入库节点仅显示绿色勾号 badge，隐藏复选框
3. 翻到章节首/末页继续滑动时自动加载上/下一章内容
4. 详情弹窗增加"删除资源"按钮（带确认）+ 快捷新建标签输入框
5. 散落文件点击后使用 FileViewer 路由，支持同目录所有文件翻页
